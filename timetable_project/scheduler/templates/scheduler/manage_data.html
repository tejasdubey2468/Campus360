Getting Started - Follow These Steps:
  1. Add Departments - Create academic departments (e.g., Computer Science, Mathematics)
  2. Add Teachers - Add faculty members and assign them to departments
  3. Add Subjects - Create course subjects with lab components if needed
  4. Add Classrooms - Set up lecture halls, seminar rooms, and laboratories
  5. Add Classes - Create student classes (e.g., CS 3rd Year A, Math 2nd Year B)
  6. Assign Subjects - Link subjects to classes with specific teachers and lecture counts
  7. Generate Timetable - Use the advanced constraint system to create optimal schedules
