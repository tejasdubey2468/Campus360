# 🚀 Campus 360 – Smart Student Management & Campus Automation System

## 📌 Overview

Campus 360 is a complete **student dashboard and campus management platform** designed to automate academic, administrative, and campus-related services in one unified system.
The platform provides students, faculty, and administrators with real-time access to academic data, attendance, resources, and campus services through a modern web interface.

This project aims to digitize campus operations and create a **smart, efficient, and centralized campus ecosystem**.

---

## 🎯 Objective

* Centralize all campus services in one platform
* Improve student and faculty interaction
* Automate attendance and academic tracking
* Provide real-time campus data and updates
* Reduce manual work in administration
* Build a scalable full-stack campus system

---

## 🧩 Key Features

### 👨‍🎓 Student Dashboard

* Student profile and academic performance
* Subject-wise section with notes and assignments
* Attendance tracking

* Faculty timetable integration
* Performance monitoring

### 👨‍🏫 Faculty Dashboard

* Faculty timetable
* QR code generation for lectures
* Attendance tracking system
* Upload notes and assignments

### 🏫 Campus Services

* Lost and Found Department
* Gymkhana and Sports Turf Booking

* Classroom and Lab Availability
* Seat/Room Occupancy Tracking
* Campus Resource Monitoring

### 🛠️ Admin Panel

* Manage students and faculty
* Control bookings and campus services
* Monitor attendance data
* Activity logs and system monitoring
* Database management

## ⚙️ Installation

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/campus360.git
```

### 2️⃣ Go to Folder

```
cd campus360
```

### 3️⃣ Install Dependencies

```
npm install
```

### 4️⃣ Run Server

```
npm start
```

### 5️⃣ Open in Browser

```
http://localhost:3000
```
